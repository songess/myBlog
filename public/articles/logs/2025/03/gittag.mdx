# 깃 태그
깃에서 다양한 명령어를 써본 거 같지만 `git tag`관련된 명령어는 써본 적이 없다. 필요성을 느낀 적도 없고.. 이번에 오픈소스수업에서 사용할 일이 있어 사용하게 되었다.

<img width={600} height={400} src="/png/gittag/250317-221201.png"/>
이러한방식으로 commit옆에 태그가 남고, 확인할 수 있다.

써야하는 상황은 특정시점을 키워드로 저장하고 싶을 때, 커밋에 버전정보를 붙이고자 할 때이다.

```shell
git tag "tag이름"
```
이런 형식으로 tag를 달면 마지막 커밋에 태그가 달린다.

```shell
git tag
```
이 명령으로 등록되어 있는 태그를 확인할 수 있고
```shell
git tag -d "tag이름"
```
이걸로 태그를 삭제할 수 있다.

<br/>`gp origin main`으로 태그도 푸시될 줄 알았지만 그렇지 않았다. `gp --tags`를 통해 태그를 푸시할 수 있었고

```shell
git push origin --delete "tag이름"
```
을 통해 원격저장소에 있는 태그를 삭제할 수 있었다.
